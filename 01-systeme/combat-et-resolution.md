## Système de jeu

Système blades in the dark en inversé : il faut que toutes les capacités aient au moins le même rang pour que la caractéristique l'ait aussi.

Exemple : Tant que la compétence 5 n'aura pas augmenté la Puissance ne passera pas à Bronze

| Puissance(Blood) | Fer | Bronze | Argent | Or  | Diamant |
| ---------------- | :-: | :----: | :----: | --- | ------- |
| Compétence 1     |  X  |   X    |        |     |         |
| Compétence 2     |  X  |   X    |        |     |         |
| Compétence 3     |  X  |   X    |   X    |     |         |
| Compétence 4     |  X  |   X    |        |     |         |
| Compétence 5     |  X  |        |        |     |         |

On lance le nombre de dés de la compétence ou de l'attribut 

- 1-3 : échec, situation empire, coût élevé pour le personnage 
- 4-5 : réussite mitigée, coût moindre
- 6+ : réussite 
- plus de 1 dé à 6 : Réussite critique
